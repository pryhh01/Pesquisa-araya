Propriedades e Atributos de Objetos

Propriedades do objeto em JavaScript
Um objeto é uma coleção não ordenada de propriedades

Propriedades são a parte mais importante dos objetos JavaScript.

As propriedades podem ser alteradas, adicionadas, excluídas e algumas são somente leitura.

Acessando propriedades JavaScript

Existem 3 maneires de sintaxe para acessar a propriedade de um objeto:


  • // objectName.property: Utilizando um "." entre o objeto e a propriedade.
    let age = person.age;

    Exemplo:
    person.firstname + " is " + person.age + " years old.";

    const person = { firstname: "John", lastname: "Doe", age: 50, eyecolor: "blue" };
    document.getElementById("demo").innerHTML = person.firstname + " is " + person.age + " years old.";
    saída: "Jonh is 50 years old."

    • Ou...

  • // objectName["property"]: Utiliza um "[]" entre o objeto e a propriedade.
    let age = person["age"];

    Exemplo:
    person["firstname"] + " is " + person["age"] + " years old.";

    • Ou...

  • // objectName[expression]: Utiliza uma variavél para chamar na espressão, x em [x].
    let age = person[x];

    Exemplo:
    let x = "firstname";
    let y = "age";
    person[x] + " is " + person[y] + " years old.";



    • Adicionando novas propriedades.

    Também é possivél adicionar novas propriedades a um objeto já existente, simplesmete atribuindo a ele um valor.


  • person.nationality = "English";
    const person = { firstname: "John", lastname: "Doe", age: 50, eyecolor: "blue" };
    person.nationality = "English";
    document.getElementById("demo").innerHTML = person.firstname + " is " + person.nationality + ".";
    saída: "John is english"

    • Excluindo propriedades.

    A palavra-chave delete excluí uma propriedade de um objeto.

  • delete person.age;
    const person = { firstName: "John", lastName: "Doe", age: 50, eyeColor: "blue" };
    delete person.age;
    Ou... excluir person["age"];

    • Obrservação!

    A palavra-chave delete excluí tanto o valor da propriedade quanto a propriedade em si.
    Após a exclusão, a propriedade não poderá ser usada antes de ser adicionada novamente.


    Objetos aninhados (agrupados dentro de outro).

    Os valores de propriedades em um objeto podem ser outros objetos.
    myObj = { name:"John", age:30,
    myCars: { car1:"Ford", car2:"BMW", car3:"Fiat" } }

    E para acessa-los você pode usar a notação "." ou "[]".

    Por exemplo:

    myObj.myCars.car2;
    myObj.myCars["car2"];
    myObj["myCars"]["car2"];

    OU associando eles a uma variavél:

    let p1 = "myCars";
    let p2 = "car2";
    myObj[p1][p2];
